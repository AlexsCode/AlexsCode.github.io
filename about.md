## This Site

This is a **markdown-driven** CV and portfolio site built with [Jekyll](https://jekyllrb.com/) and hosted on GitHub Pages.

- **CV** — populated from `_data/experience.yml`, `_data/skills.yml`, `_data/education.yml`
- **Portfolio** — project pages live in `_portfolio/`
- **Blog** — posts in `_posts/`

Everything is plain Markdown. No database, no CMS, no JavaScript framework — just static HTML generated from `.md` files.
